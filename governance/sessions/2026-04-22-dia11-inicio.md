## Riesgos conocidos

1. **Secret keys ausentes:** si no tenemos las credenciales actuales, necesitamos regenerar
2. **Socket Mode conexión:** primera vez puede tomar 30-60s conectar
3. **ECS task restart loop:** si secret está mal, task reinicia constantemente
4. **Costo inicial:** primeras horas pueden generar ~$0.01-0.02 hasta optimizar

## Rollback plan

Si algo falla:
```bash
# Delete stack (destruye recursos, preserva logs)
aws cloudformation delete-stack --stack-name atlas-platform-prod

# Cleanup manual si es necesario
aws ecr delete-repository --repository-name nova-atlas --force
aws secretsmanager delete-secret --secret-id nova/atlas/felirni/config --force-delete-without-recovery
```
